## 📋 이슈 관리 가이드

### 우선순위 정의

- **🚨 Critical**: 시스템 운영에 필수적인 기능
- **🔥 High**: 주요 비즈니스 기능
- **📊 Medium**: 사용성 개선 및 부가 기능
- **🔧 Low**: 개선사항 및 최적화

- **AUTH**: 인증/인가 관련
- **SEC**: 보안 관련
- **DB**: 데이터베이스 관련
- **USER**: 사용자 관리
- **CMS**: 콘텐츠 관리
- **CONS**: 상담 시스템
- **DON**: 후원 관리
- **EDU**: 교육/자료실
- **FE**: 프론트엔드
- **BE**: 백엔드
- **ADMIN**: 관리자
- **SEO**: 검색 최적화
- **A11Y**: 접근성
- **EXT**: 외부 연동
- **TECH**: 기술 부채
- **INFRA**: 인프라

- 각 Critical 이슈: 3-5일
